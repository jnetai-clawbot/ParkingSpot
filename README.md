# ParkingSpot — Parking Spot Finder & Tracker

**Job 60** | **Package:** com.jnetai.parkingspot

## What It Does
Find and track parking spots — save favourite locations, track availability, record costs, and get reminders when time is running out.

## Features
- Save parking locations with notes
- Track parking session timer
- Cost calculator (hourly/daily rates)
- Favourite spots list
- Parking history with stats
- About page with version and update check
- Data export as JSON

## Build
- Android API 26+, Kotlin, Room, KSP, ViewBinding + XML, Material3 Dark
